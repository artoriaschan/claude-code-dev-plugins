# Demo Skill

This is an example skill demonstrating how to write a Claude Code skill. It serves as a reference for plugin developers.

## Trigger

User asks about skill structure, or mentions "demo skill" or "example skill".

## Purpose

This skill shows the proper format and structure of a Claude Code skill, including:
- YAML frontmatter fields
- Trigger description format
- Markdown body with instructions
- How to reference supporting files

## Example Usage

This skill auto-activates when the user asks about skill structure or mentions "demo skill" or "example skill".

This skill is part of the example-plugin template, showing how skills integrate into a plugin marketplace project.
