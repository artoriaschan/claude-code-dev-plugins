# 插件 README 模板

为每个插件创建 `README.md`，遵循官方插件格式。
从 `git config user.name` 和 `git config user.email` 获取作者信息。

```markdown
# <Plugin Name>

<一行描述>

## What It Does

<插件提供的内容简述>

## Usage

<示例触发短语或使用场景>

## Components

| 类型 | 名称 | 用途 |
|------|------|---------|
| Skill | `<skill-name>` | <Skill 用途> |
| Agent | `<agent-name>` | <Agent 用途> |

仅包含存在的组件行。

## Authors

<git user.name> (<git user.email>)
```
