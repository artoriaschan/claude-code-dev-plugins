# Plugin README Template

Create `README.md` for each plugin following the official plugin format.
Get author info from `git config user.name` and `git config user.email`.

```markdown
# <Plugin Name>

<One-line description>

## What It Does

<Brief description of what the plugin provides>

## Usage

<Example trigger phrases or usage scenarios>

## Components

| Type | Name | Purpose |
|------|------|---------|
| Skill | `<skill-name>` | <Skill purpose> |
| Agent | `<agent-name>` | <Agent purpose> |

Only include rows for components that exist.

## Authors

<git user.name> (<git user.email>)
```
