# cli-builder

## 用途

TypeScript CLI 项目脚手架和命令模板生成。提供基于 pnpm 的 monorepo CLI 项目交互式脚手架，包含 Commander、Zod、Chalk 和完整工具链。

## 组件

| 类型 | 路径 | 用途 |
|------|------|------|
| 清单 | `.claude-plugin/plugin.json` | 插件名称、版本、描述 |
| Skill | `skills/cli-create/SKILL.md` | 交互式 CLI 项目创建，包含 TypeScript ESM、Commander、Zod、Chalk 和完整工具链 |
| Skill | `skills/cli-add-command/SKILL.md` | 生成命令模板文件，包含 Commander 定义、Zod 验证、Chalk 输出 |
