# cli-builder

## Purpose

TypeScript CLI project scaffolding and command template generation. Provides interactive scaffolding for creating pnpm-based monorepo CLI projects with Commander, Zod, Chalk, and full tooling.

## Components

| Type | Path | Purpose |
|------|------|---------|
| Manifest | `.claude-plugin/plugin.json` | Plugin name, version, description |
| Skill | `skills/cli-create/SKILL.md` | Interactive CLI project creation with TypeScript ESM, Commander, Zod, Chalk, and full tooling |
| Skill | `skills/cli-add-command/SKILL.md` | Generate command template files with Commander definition, Zod validation, Chalk output |
